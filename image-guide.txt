RECOMMENDED DIMENSIONS:
- Hero Images: 1920x1080px (JPG, compressed)
- Product Images: 800x1000px (JPG, 4:5 aspect ratio)
- Square Assets: 800x800px (JPG)
- Logos/Icons: Transparent PNG or SVG

INSTRUCTIONS FOR REPLACEMENT:
1. Prepare your image with the recommended dimensions.
2. Name the file exactly as listed above (e.g., 'product-1.jpg').
3. Upload the file to the correct subfolder within '/images/'.
4. The website will automatically update with the new image.
